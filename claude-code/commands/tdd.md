# TDD Command

Execute the Test-Driven Development workflow.

## Usage

Type `/tdd` in Claude Code chat to start the TDD workflow.

## Workflow

1. **RED**: Write a failing test that describes the desired behavior
2. **GREEN**: Write minimal code to make the test pass
3. **REFACTOR**: Improve code quality while keeping tests green

## Requirements

- Test coverage: 80%+
- One test at a time
- Fast feedback loop
- Clean, readable tests

## Example

```
User: /tdd Implement user authentication

Agent: 
1. Let's start with RED phase - write a failing test for user login
2. Then GREEN phase - implement minimal code
3. Finally REFACTOR - improve the code
```

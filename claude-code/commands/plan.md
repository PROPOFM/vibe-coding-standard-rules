# Plan Command

Create an implementation plan for a feature or task.

## Usage

Type `/plan` in Claude Code chat followed by the feature description.

## Workflow

1. Analyze requirements and current codebase
2. Break down into actionable steps
3. Identify dependencies and risks
4. Present plan for user approval

## Example

```
User: /plan Add user profile editing feature

Agent:
## Implementation Plan: User Profile Editing

### Steps
1. Create profile edit form component
2. Add API endpoint for profile updates
3. Implement validation
4. Add tests
...
```

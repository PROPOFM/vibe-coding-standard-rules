# 作業記録（History）

このフォルダは、タイムスタンプごとの作業記録を管理します。

## 目的

- 日々の作業内容の記録
- 決定事項の記録
- 問題と解決策の記録
- 会議の議事録

## ファイル命名規則

- `[YYYYMMDD]_[summary].md`: 日付と概要で命名
- 例: `20240115_daily-standup.md`, `20240116_architecture-decision.md`

## 推奨フォーマット

```markdown
# [日付] [概要]

## 作業内容
- [x] タスク1: [完了内容]
- [ ] タスク2: [進行中]
- [ ] タスク3: [未着手]

## 決定事項
- 決定1: [内容]
  - 決定理由: [理由]
  - 影響範囲: [範囲]
- 決定2: [内容]
  - 決定理由: [理由]
  - 影響範囲: [範囲]

## 課題・問題
- 課題1: [内容]
  - 解決策: [内容]
  - ステータス: [解決済み/進行中/未解決]
- 課題2: [内容]
  - 解決策: [内容]
  - ステータス: [解決済み/進行中/未解決]

## 学んだこと
- 学び1: [内容]
- 学び2: [内容]

## 次回の予定
- [ ] 次回タスク1
- [ ] 次回タスク2
- [ ] 次回タスク3

## 参考資料
- [リンク1](url)
- [リンク2](url)
```

## 運用方針

### 日次記録
- 毎日の作業終了時に記録
- 完了したタスク、進行中のタスク、課題を記録

### 週次記録
- 週の終わりに週次サマリーを作成
- 週の成果、課題、次週の計画を記録

### 重要決定の記録
- アーキテクチャの決定
- 技術選定の決定
- プロセス変更の決定

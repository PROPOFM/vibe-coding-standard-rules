# 要件検討（Consideration）

このフォルダは、機能や要件を検討する段階の情報を管理します。

## 目的

- 機能の検討事項の記録
- 複数の選択肢の比較
- 検討中の課題の整理
- 意思決定の記録

## ファイル命名規則

- `[feature-name].md`: 機能名で命名
- 例: `user-authentication.md`, `payment-integration.md`

## 推奨フォーマット

```markdown
# [機能名] 要件検討

## 背景・目的
機能を検討する背景と目的

## 検討事項
- 検討事項1: [内容]
- 検討事項2: [内容]

## 選択肢の比較

### Option 1: [選択肢名]
- メリット: [内容]
- デメリット: [内容]
- 実装難易度: [評価]

### Option 2: [選択肢名]
- メリット: [内容]
- デメリット: [内容]
- 実装難易度: [評価]

## 決定事項
- 選択した選択肢: [Option X]
- 決定理由: [理由]

## 参考資料

- [リンク2](url)
```
